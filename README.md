# 模擬試験結果閲覧アプリ

柔道整復師の模擬試験結果を閲覧できる学生用アプリケーションです。

## 機能

- 総合点、必修、一般合計の表示
- 各科目（解剖学、生理学、運動学、病理学、衛生学、リハビリ医学、一般臨床、外科学、整形外科、柔整理論）の成績表示
- 必修および全科目の順位、偏差値、得点率の表示
- 複数の学生データの切り替え表示

## セットアップ

1. 依存パッケージのインストール:
```bash
npm install
```

2. 開発サーバーの起動:
```bash
npm run dev
```

3. ブラウザで http://localhost:5173 を開く

## ビルド

本番用ビルド:
```bash
npm run build
```

ビルドしたファイルのプレビュー:
```bash
npm run preview
```

## データのカスタマイズ

`src/sampleData.ts` ファイルで学生のサンプルデータと満点設定を編集できます。

## 技術スタック

- React 18
- TypeScript
- Vite
